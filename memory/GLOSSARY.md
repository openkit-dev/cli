# Glossary

| Term | Definition |
|---|---|
| OpenKit Runtime | Rust CLI implementation located in `rust-cli/`. |
| Memory Kernel | Persistent documentation and operational context model under `.openkit/` and `openkit-memory/`. |
| Contract Test | Integration test validating command behavior and generated artifacts (`rust-cli/tests/command_contracts.rs`). |
| Discovery Gate | Mandatory context-generation phase before `/specify`. |
| Release Matrix | Multi-platform build target list in `.github/workflows/release.yml`. |
| Drift | Mismatch between docs/process references and real repository state. |
